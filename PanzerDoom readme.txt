--------System requirements

Operating system: Windows XP or later.
CPU: pentium 4 or better.


--------Most useful command line arguments

-game [game name]
    Select game. Available "shareware_doom", "doom", "ultdoom", "doom2", "tnt", "plutonia".

-iwad [file]
    Select IWAD for game.

-waddir [dir]
    Select directory for searching of IWAD files.

-file [file1] [file2] ...
    Add additional PWAD file(s).

-config [file]
    Select custom config file. Default is ./default.cfg


--------Most usefull config variables

"fullscreen" 0/1
    Run in fullscreen or windowd mode.

"v_32bit" 0/1
    Use 32bit renderer, or vanila 8bit renderer.

"v_scaler" number
    Game framebuffer pixel size. Use 1 for better quality, use 2 or more for "oldschool-style".

"screenwidth"
"screenheight"
    Screen resolution on fullscreen mode or inner window size in windowed mode.

"use_mouse" 0/1
    Use mouse or not for control.

"freelook" 0/1
    Use free look or not. Note: free look works only in 32bit renderer and does not affect game logic, autoaim still works.
